# PlanetwideShield 全球护盾

Dyson Sphere Program 的 BepInEx 模组：当同一星球上有**两座护盾塔彼此挨着**时，自动将该星球的护盾覆盖改为**全球**。

## 功能

- **自动判定**：游戏对护盾塔有最小放置距离（由建筑 PrefabDesc.roughRadius 决定）。若两座护盾塔中心距离小于 **1.5 × 该最小距离**，则视为“挨着”，该星球护盾按全球覆盖计算。
- **无 UI、无存档**：不增加开关，不写入任何配置或存档；仅根据当前护盾塔位置实时判定。

## 使用

在已有至少两座护盾塔的星球上，将其中两座建得足够近（小于游戏允许最小间距的约 1.5 倍），该星球护盾即自动变为全球覆盖。

## 说明

- 仅改变护盾的**覆盖形状**（通过虚拟生成点让 Shader 算出全球覆盖），能量、伤害等仍使用游戏原有逻辑。
- 最小距离取自护盾塔 Prefab 的 `roughRadius`（与建造碰撞一致）；若取不到则用默认值 2，阈值约为 6 格。
